# Compute Summary Statistics for Numeric Variables

Compute Summary Statistics for Numeric Variables

## Usage

``` r
cb_summary_stats_numeric(df, .x, digits = 2)
```

## Arguments

- df:

  Data frame of interest

- .x:

  Column of interest

- digits:

  Number of digits after decimal to display

## Value

A tibble
