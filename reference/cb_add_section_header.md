# Create Formatted Section Header

Create Formatted Section Header

## Usage

``` r
cb_add_section_header(rdocx, text = NA)
```

## Arguments

- rdocx:

  rdocx object created with
  [`officer::read_docx()`](https://davidgohel.github.io/officer/reference/read_docx.html)

- text:

  Text of section header

## Value

rdocx object
